# Vehicle DOD :taco:

## Desc
This repository contains 3 classes:
- Vehicle class
- Car class
- Plane class
The car and plane classes are both child classes of the vehicle class.

## Characteristics

### Vehicle
- n_passengers
- size_cargo

### Car
- brand
- horse_power
- max_speed

### Plane
- Inherited from Vehicle class
